# Project Title

React Blog Website Design


## Run Locally

Clone the project

```bash
  git clone https://github.com/OrhanEmirhanUlkatan/react-blog-website-desing.git
  
```

Go to the project directory

```bash
  cd react-blog-website-desing
```

Install dependencies

```bash
  npm install || yarn 

```

Start the server

```bash
  npm start || yarn start
```
